# Simple Clock App

---

## 📝 Description

A straightforward clock application that displays the current time, day of the week, and date.

#### 💻 Technologies

- HTML
- CSS
- JavaScript

---

## ✨ Features

- Displays the current time.
- Displays the current weekday.
- Displays the current date.
